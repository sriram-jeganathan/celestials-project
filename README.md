# Celestials Project

## Install NodeJS in Ubuntu
```
sudo apt update
sudo apt install -y nodejs
node -v
```

## Install SQLite in Ubuntu
```
sudo apt update
sudo apt instal sqlite3
sqlite3 --version
```
